UsiWSC project overview

UsiWSC is a framework for generation of a usable user interface for Interactive Web Service Composition. UsiWSC allows:

  • the expression of the user interactions in a Web service composition;
  • the generation of an abstract user interface in order to support: multi-users, multi-context and multi modalities;
  • the management of the consistency between the control flow of the composition and the navigation flow of its user interface;
  • the framework is fully compliant with all BPEL engines and UsiXML tools. more...
